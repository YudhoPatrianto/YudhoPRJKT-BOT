# YudhoPRJKT-BOT
The Telegram Bot To Send Files And Notify 


#### Usage 

## Input Your Auth Bot Token And Chat ID First
---------------
```bash
nano .secret/token
```

## And 
---------------
```bash
nano .secret/chatid
```

### Give The Bot To Full Access Control File
---------------
```bash
chmod 777 run_bot.sh
```

# Run The Bot
---------------
```bash
bash run_bot.sh
```
